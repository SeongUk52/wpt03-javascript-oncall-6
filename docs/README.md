# 미션 - 개발자 비상근무

## 입출력
- [ ] 월과 시작요일을 입력받는다.
- [ ] 평일 및 휴일 근무순서를 입력받는다.
- [ ] 근무표를 날짜 순서대로 출력한다.

## 검증
- [ ] 월과 시작요일이 유효한 입력인지 검증한다.
- [ ] 평일 또는 휴일에 같은 근무자가 2번 이상 편성되어있지 않은지 검증한다.
- [ ] 근무자명이 5글자 이하인지 검증한다.
- [ ] 근무자 수가 5명 이상 35명 이하인지 검증한다.
- [ ] 평일근무자 목록과 휴일근무자 목록이 동일한지 검증한다.
- [ ] 올바르지 않은 입력을 할 경우 `[ERROR]`로 시작하는 에러 메시지를 출력 후 다시 입력받는다.

## 근무순서
- [ ] 큐 자료구조를 사용하여 근무자의 근무 순서가 순환되도록 한다.


## 근무일
- [x] 각 근무일에는 한 명의 비상 근무자가 배정되며, 요일을 기억한다.
- [x] 휴일목록과 평일목록을 뽑아낸다.
- [ ] 해당 근무일이 이번달의 말일인지 판단한다.
- [ ] 연속된 근무자가 있을 경우 앞에서부터 근무자의 순서를 변경한다.
